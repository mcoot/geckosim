# 0001 — Project vision

- **Status:** Accepted
- **Date:** 2026-04-26

## Context

Establishing what we're building before any architecture choices.

## Decision

A simulation in the spirit of The Sims, SimCity, and Dwarf Fortress, where the agents are geckos. Geckos are mostly thematic — for simulation purposes they behave as humans in a modern society, with some aesthetic gecko-specific flavor.

Scope spans both:

- **Individual agents** — needs, jobs (accountants, janitors, …), daily lives.
- **Macro society** — crime, disasters, housing policy, economy.

## Notes

Not a "game" in the traditional sense — more an interactive simulation. The player interaction model is open and to be designed later.
